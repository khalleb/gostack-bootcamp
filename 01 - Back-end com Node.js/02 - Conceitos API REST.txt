
* Uma requisição feita por um cliente
* Resposta retornada através de uma estrutura de dados

* Métodos HTTP/Rotas
  - GET - Obter informação
  - POST - Criar uma nova informação
  - DELETE - Deletar uma informação
  - PUT -> Alterar uma informação

* Benefícios
  - Mútiplos clientes
  - Protocolo de comunicação padronizada
  - Usa o JSON(JavaScript Object Notation) como um protocolo global de comunicação

* Conteúdo de requisição
  - Route
  - Route param
  - Query params
  - Body

* HTTP Codes
  - 1xx - Informational
  - 2xx - Success
  - 3xx - Redirection
  - 4xx - Client Error
  - 5xx - Sever Error
